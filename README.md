# Fitness Journey

A personal running journal documenting my journey of consistency, discipline, and building a healthier lifestyle through running.

## Live Site

View the live journal at: [https://skumar2112.github.io/fitnessjourney/](https://skumar2112.github.io/fitnessjourney/)

## About

This site tracks my running progress over time, including stats, reflections, and milestones. It's built as a simple, public record of showing up consistently.

## Features

- Monthly running stats and charts
- Personal reflections and learnings
- Interactive dashboard for different time periods
- Public accountability journal
